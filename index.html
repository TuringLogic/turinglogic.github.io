Adding Device Information for Custom Boards in the Base System Builder
If you selected the Custom Board option (as described in Selecting a Target Development Board), you must provide information about peripherals to the Base System Builder. You can do the following in the Peripheral Configuration page of the BSB.
Adding a Peripheral Device
  1. Click Add Device.
  2. In the Add Devices dialog box that opens, select the device information.
  3. If you want to add the device to your system in addition to the Peripheral list, click the Add it to the system check box. Otherwise, BSB only adds it to the Peripheral list.
  4. Click OK. BSB adds the device.
Exporting Devices
To avoid re-adding the devices for your board in later projects, you can export the device list and import it into other projects. To do this:
  1. After adding the devices for your board, click Export in the Peripheral Configuration page of the BSB. The Export Device List dialog box opens.
  2. Type a name for the file, or click Browse to specify a location and existing file. You can review the list of devices that will be included in the file.
  3. Click OK. BSB creates the file.
Importing Devices
To import the file into a later project:
  1. In the Peripheral Configuration page of the BSB, click Import. The Import Device List dialog box opens.
  2. Click Browse, then browse to and select the device list .txt file.
  3. Click OK. BSB imports the device list into your project.
